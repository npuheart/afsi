## 通过 Nanobind 使用C++代码
Installation
------------

1. Clone this repository
2. Run `pip install .`

Afterwards, you should be able to issue the following commands (shown in an
interactive Python session):

```pycon
>>> import afsic
>>> afsic.add(1, 2)
3
```
